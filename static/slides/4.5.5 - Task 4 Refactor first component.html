Task 4: Refactor into two components

  1. Create a new directory in your project: react-intro/src/components
  2. In the directory, create a new file: react-intro/src/components/ColleagueList.js
  3. In the file, define a function component named ColleagueList
  4. Copy-and-paste the hard-coded list from App into ColleagueList
  5. Import ColleagueList into: react-intro/src/App.js
  6. Replace the hard-coded list in App with an instance of ColleagueList
  7. The app should now look identical to how it was before üòé
Nice to know
  • Reusable components are typically placed into src/components
    • Any components may consist of reusable components
    • Reusable components only consist of reusable components
  • .jsx extension may be used for files containing JSX
  • .js(x) extensions are not required in import paths
    • If webpack is configured to resolve the extensions
    • create-react-app automatically fixes this
