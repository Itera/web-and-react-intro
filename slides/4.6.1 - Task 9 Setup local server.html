Task 9: Setup local server

  1. Run following command in a directory outside of your React project: git clone https://github.com/Itera/web-and-react-intro.git
  2. Navigate into the cloned project directory: cd web-and-react-intro
  3. Install the project's NPM dependencies: npm install
  4. Start the local server by running the following command: npm run server
  5. Check server by navigating to the following URL in your browser: http://localhost:4000/colleagues
Nice to know
  • Web browser use the Hypertext Transfer Protocol (HTTP)
    • Requests web pages or raw data using the GET method
    • Submits form data using the POST method
  • REST APIs typically makes use of many HTTP methods
    • E.g. PUT and DELETE
