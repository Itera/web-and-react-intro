Nice to know
  • Reusable components are typically placed into src/components
    • Any component may use a reusable component
    • Reusable components should not use non-reusable component
  • .jsx extension may be used for files containing JSX
  • .js(x) extensions are not required in import paths
    • If webpack is configured to resolve the extensions
    • create-react-app automatically fixes this
