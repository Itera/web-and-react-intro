Why use React?

  • Easier mental model
    • One-way, immutable, data flow
      • More predictable
      • Easier to reason about
  • Reusability
    • Encapsulate HTML, CSS, and JavaScript into components
    • Compose components into something bigger
  • A library, not a framework
    • Doesn't drastically change the way you write JS
    • If you know React you know JS
