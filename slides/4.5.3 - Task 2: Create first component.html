Task 2: Creating your first component

  1. Remove everything from the file: react-intro/src/App.js
  2. In that file, define a function component named App with:
    • a heading that reads: "My Colleagues"
    • a hard-coded table that lists out your colleagues:
      • name
      • education/background
      • home town
  3. Export your function component as a default export
  4. Yay! 🥳 You've created your first component 🤩
Nice to know
  • The App component is your root component
    • Mounted to DOM in react-intro/src/index.js
  • Keep the dev-server running while you program
    • Hot Reloading automatically updates app in browser
  • React will complain about missing <thead> and <tbody> elements
    • Check the browser console
    • Look them up: thead and tbody
    • Not required in HTML5
