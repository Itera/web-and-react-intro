Task 1: Creating a React project using create-react-app

  1. Open your terminal
  2. Navigate to a directory of your choosing
  3. Run command: npx create-react-app react-intro
  4. Press ENTER/RETURN when prompted with: Ok to proceed? (y)
  5. Congratulations, you should now have a directory called react-intro ðŸŽ‰. This is your React project.
Nice to know
  • Check out the starter app by running the following command in your project directory: npm start

    • It serves the React app locally on your machine
    • By default, the app is only accessible by you
