Elements

  • Atomic building blocks
  • Represents HTML elements
  • Instantiated using React.createElement
    • 1st parameter: HTML element name
    • 2nd parameter: object of HTML element attributes
      • children is an attribute in React
  • JSX is syntactic sugar for instantiation
    • Attribute in camelCase (not lowercase)
    • Attribute can accept any JS value
    • Attribute are called props instead

          import React from 'react';

          const elem = React.createElement(
            'h1',
            {
              styles: { color: 'red' },
              children: ['This is a heading']
            }
          );
