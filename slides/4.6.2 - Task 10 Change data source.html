Task 10: Change data to live in server

Change your app to:

  1. Fetch colleagues from the server to populate the list
  2. Submit new colleagues to the server
  3. Submit changes to existing colleagues to the server
  4. Delete colleagues from the server
  5. Always stay up-to-date with the information on the server
Tip #1

useEffect with an empty dependency list mimicks the componentDidMount life-cycle method.

Nice to know
  • fetch as an in-built function (ES6+)
    • Sends an HTTP request
    • Wrapper around XMLHttpRequest
  • HTTP requests are expensive
    • Avoid unnecessary requests
    • Correct usage of dependency lists
